## Projects

### Speech Bridge - live captions during lectures at PJATK *(closed-source)*

Leading a Student Government project that aims to help increase the reach of lectures at PJATK by implementing live captions displayed in English in the lecture hall. Because this project is a collaboration with the university, it has to remain closed-source for now.

`React` `Typescript` `Electron` `Cloudflare Workers` `Deepgram` `DeepL`

### [Simple Stocks](https://github.com/kuba-walczak/simple-stocks) - stock tracking app built in two versions: for the web and mobile

A personal stock tracking app available in two versions - a web app built with Next.js and a native mobile app built with React Native (Expo). Both share the same Supabase backend and Finnhub data source, giving users the same experience across platforms.

`React` `Typescript` `Next` `Expo` `Supabase`

### [Portfolio](https://github.com/kuba-walczak/portfolio) - personal website where I showcase my creative and technical work

Built with Next.js, React Three Fiber, and GSAP. The site features an interactive 3D hero section and smooth scroll-driven animations. All media is fetched from a CDN.

`React` `Typescript` `Next` `GSAP` `Three` `R3F`
